# DEPRECATED

This directory (`agents/commands-lite/`) contains the **lite pipeline** command set and is **no longer maintained**.

## What replaced it

The active source-of-truth is now `/skills/<category>/<slug>/SKILL.md` at the repo root, following the `mattpocock/skills` layout convention. The v2 pipeline lives at `/skills/SDD/`.

- See [`/INSTALL.md`](../../INSTALL.md) for install patterns
- See [`/AGENTS.md`](../../AGENTS.md) for the new skill catalog

## What happens to these files

The lite pipeline was a parallel cohort kept in sync with the full pipeline. It has been superseded by the v2 skills. **Planned for deletion in a future cleanup pass.** Do not add new commands here.
